# のっかれボールズ v0.2.2

スマホブラウザで遊べる、押し出し対戦ミニゲームのプロトタイプです。  
マリオパーティのミニゲームをそのまま再現するのではなく、オリジナルの「ボールに乗って押し合うゲーム」として作っています。

## 現在できること

- スマホ1台で2人対戦
- スマホ縦画面で 1P スティックを上、2P スティックを下に配置
- PCでは 1P: WASD / 2P: 矢印キー
- 円形ステージ
- 慣性移動
- 円同士のインパルス計算による弾性衝突
- ぶつかった方向のベクトルを記録し、逆方向入力でその慣性だけを減少
- 場外落下判定
- TIME 30 / 60 / 90 / ∞ 設定
- STOCK 1 / 2 / 3 / 5 設定
- ストック制の復帰
- 勝敗表示

## 遊び方

`index.html` をブラウザで開くだけで遊べます。  
GitHub Pages / Cloudflare Pages にもそのまま置けます。

## v0.2.2の修正

- 衝突を「押し合い補正」から、円同士の弾性衝突インパルス計算に変更
- ぶつかった時に、より分かりやすく弾くように調整
- 中央方向入力による端の補助を廃止
- 最後に受けた衝突ベクトルを保存し、その反対方向に入力した時だけ慣性を削る方式へ変更
- 速度 / 加速 / 減速の操作感は v0.2.1 を維持
- 反射強化に合わせて、衝突後の最大速度・場外猶予を再調整

## 調整メモ

反射が強すぎる場合は `game.js` の以下を下げます。

```js
restitution: 0.82,
postCollisionMaxSpeed: 430,
minBounceImpulse: 58,
```

逆入力で踏ん張れすぎる場合は以下を下げます。

```js
vectorBrakeAccel: 1120,
vectorBrakeDrag: 3.2,
vectorBrakeDuration: 0.82,
```

端でまだ落ちやすい場合は、逆に上げます。
