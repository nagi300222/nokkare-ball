# のっかれボールズ v0.2.1

スマホブラウザで遊べる、押し出し対戦ミニゲームのプロトタイプです。
マリオパーティのミニゲームをそのまま再現するのではなく、オリジナルの「ボールに乗って押し合うゲーム」として作っています。

## 現在できること

- スマホ1台で2人対戦
- スマホ縦画面で 1P スティックを上、2P スティックを下に配置
- PCでは 1P: WASD / 2P: 矢印キー
- 円形ステージ
- 慣性移動
- ボール同士の反射つき衝突
- 端で中央方向へ入力すると踏ん張りやすい補助
- 場外落下判定
- TIME 30 / 60 / 90 / ∞ 設定
- STOCK 1 / 2 / 3 / 5 設定
- ストック制の復帰
- 勝敗表示

## 遊び方

`index.html` をブラウザで開くだけで遊べます。
GitHub Pages / Cloudflare Pages にもそのまま置けます。

## ファイル構成

```text
index.html   画面構造
styles.css   スマホ向けUI
game.js      ゲーム本体
README.md    この説明
ONLINE_PLAN.md 通信対戦化の設計メモ
```

## v0.2.1の修正

- v0.2.0の上下スティック / TIME∞ / STOCK設定は維持
- 速度・加速・減速の操作感をv0.1.1相当に戻す
- 反射を弱め、端で中央方向へ入力していれば踏ん張りやすい強さに再調整
- 衝突後の最大速度を抑えて、理不尽な一撃落下を減らす
- 衝突時の画面揺れを少し控えめに調整

## 調整メモ

反射が強すぎる場合は `game.js` の以下を下げます。

```js
restitution: 0.56,
postCollisionMaxSpeed: 405,
```

端で粘りすぎる場合は以下を下げます。

```js
edgeSaveStrength: 0.62,
edgeInwardAssist: 470,
```

端でまだ落ちやすい場合は、逆に上げます。
