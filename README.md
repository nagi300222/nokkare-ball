# のっかれボールズ v0.2.0

スマホブラウザで遊べる、押し出し対戦ミニゲームのプロトタイプです。
マリオパーティのミニゲームをそのまま再現するのではなく、オリジナルの「ボールに乗って押し合うゲーム」として作っています。

## 現在できること

- スマホ1台で2人対戦
- スマホ縦画面で 1P スティックを上、2P スティックを下に配置
- PCでは 1P: WASD / 2P: 矢印キー
- 円形ステージ
- 慣性移動
- ボール同士の反射つき衝突
- 端で中央方向へ入力すると踏ん張りやすい補助
- 場外落下判定
- TIME 30 / 60 / 90 / ∞ 設定
- STOCK 1 / 2 / 3 / 5 設定
- ストック制の復帰
- 勝敗表示

## 遊び方

`index.html` をブラウザで開くだけで遊べます。
GitHub Pages / Cloudflare Pages にもそのまま置けます。

## ファイル構成

```text
index.html   画面構造
styles.css   スマホ向けUI
game.js      ゲーム本体
README.md    この説明
ONLINE_PLAN.md 通信対戦化の設計メモ
```

## v0.2.0の修正

- スマホ画面のスティックを上下配置に変更
- TIME 設定に `∞` を追加
- STOCK 設定を追加
- ストックが残っている場合は落下後に復帰するよう変更
- ぶつかったときに速度が反射する衝突処理へ変更
- 端で反対方向、つまり中央方向へ入力していれば落ちにくくなる補助を追加
- ステージ表示が縦長画面で楕円にならないよう、Canvas描画を等倍スケールに修正
- キーボード入力を離した時に方向入力が残りにくいよう入力管理を分離

## 調整メモ

反射が強すぎる場合は `game.js` の以下を下げます。

```js
restitution: 0.78,
postCollisionMaxSpeed: 430,
```

端で粘りすぎる場合は以下を下げます。

```js
edgeSaveStrength: 0.62,
edgeInwardAssist: 470,
```

端でまだ落ちやすい場合は、逆に上げます。
